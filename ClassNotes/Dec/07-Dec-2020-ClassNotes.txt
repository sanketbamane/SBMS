Yesterday's session : API Gateway & Request Routings

-----------------------------------------------------------------

-> API Gateway is used to manage all the apis which are available in our application.

-> It acts as front door for accessing apis of an application

-> Spring Cloud Netflix libraries provided Zuul proxy as an api gateway.

-> Google provided 'Apigee' as a commericial api gateway

-> In API Gateway we will configure Routings. When request recieved by Api gateway it will route that request to particular api available in application based on URL mapping.
